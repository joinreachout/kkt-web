# Retail AI Canvas — the open playbook

External: **retail.kittykat.tech**

## What it is

A full open playbook of where AI and PMO discipline create measurable
value across a mid-sized retailer.

- **30 services** across **11 retail domains** (forecasting,
  replenishment, supplier ops, loss prevention, customer/loyalty,
  pricing, store ops, warehouse/transport, e-commerce, claims, AI
  workforce assistants, etc.)
- Built from real engagements
- Updated quarterly
- Has its **own advisor** that walks the user through it
  (retail.kittykat.tech advisor — different surface, deeper retail
  knowledge than this one)

## When to send the user there

- They ask about retail-specific topics that benefit from the canvas
  card-level detail (specific replenishment patterns, loyalty
  segmentation, supplier service control, etc.)
- They want to browse "what could AI realistically do in my retail
  business" — that's the playbook's job
- They want a self-serve diagnostic (the canvas has a 3-question
  starter that maps pressures to recommended services)

## When this advisor handles the question directly

- High-level retail strategy questions where the canvas is too granular
- KKT-as-firm questions ("could you help me with X?")
- Questions about Optimus, fuel networks, our other cases
- Drop-doc audits

## Roadmap layer (for sequencing questions)

The canvas uses a maturity progression for sequencing recommendations:

- `see` — make current state visible
- `control` — get exception handling and ownership in place
- `optimize` — make better decisions on the controlled foundation
- `scale` — embed the routine across teams
- `expand` — new value pools beyond the core

Default sequencing logic:
- `see` before `control`
- `control` before `optimize`
- `optimize` before `scale`
- `scale` before `expand`

Most clients enter at `see` or `control`. Stage `expand` should
rarely be the first recommendation.

## Stage layer (for browsing)

The canvas is organized in three stages:

- **Stage 1 — Strengthen Core Operations**
- **Stage 2 — Optimize and Grow**
- **Stage 3 — Expand and Monetize**

Stages are for how the canvas is browsed. Roadmap steps are for
sequencing. Don't conflate them.
